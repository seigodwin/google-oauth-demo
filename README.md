# Google OAuth Authentication & Authorization Demo

A practical ASP.NET Core Web API project demonstrating how to implement **authentication and authorization using external identity providers**, with Google OAuth as the provider.

The goal of this project is to understand how modern applications allow users to authenticate using existing accounts from trusted providers like Google instead of creating and managing separate usernames and passwords.

## 🚀 Features

- Authentication using Google OAuth 2.0
- External identity provider integration
- Cookie-based authentication
- Authorization using ASP.NET Core `[Authorize]` attribute
- Protected API endpoints
- Environment variable configuration for sensitive credentials
- OpenAPI documentation with Scalar

## 🛠️ Technologies Used

- ASP.NET Core Web API
- C#
- .NET 10
- Google OAuth 2.0
- ASP.NET Core Authentication & Authorization
- Scalar OpenAPI UI
- DotNetEnv for environment configuration
